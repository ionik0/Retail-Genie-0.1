## âš¡ Troubleshoot

| Problem | Solution |
|---------|----------|
| "Can't reach service" | Check all 3 services running on correct ports |
| "No recommendations" | Verify recommender has products.json loaded |
| "CORS error" | Check VITE_ORCHESTRATOR_URL in .env.local |
| "Module not found" | Run `npm install` and `pip install -r requirements.txt` |
